import { Button } from './button';

## React Component for rendering text

A basic button 
### Component usage
```js
<Button>Click me!</Button>
```

### Type

There are `primary` buttons, `default` buttons, `text` buttons, and `link` buttons.
```js live
<>
  <Button>Default Button</Button>
  <Button type='primary'>Primary Button</Button>
  <Button type='text'>Text Button</Button>
  <Button type='link'>Link Button</Button>
</>
```

### Danger Buttons

```js live
<>
  <Button danger>Default</Button>
  <Button danger type='primary'>Default</Button>
  <Button danger type='text'>Text</Button>
  <Button danger type='link'>Link Button</Button>
</>
```
